A web-based application designed to help users create professional resumes in both ATS-friendly and modern visually appealing formats.

The platform allows users to input their personal, educational, and professional information through a structured and user-friendly interface. It then dynamically generates two types of CVs:

ATS Format: Optimized for Applicant Tracking Systems using clean structure, proper keyword placement, and standardized sections to ensure high compatibility with automated recruitment software.
Modern Format: Designed with visually engaging layouts, suitable for human recruiters, featuring enhanced styling, colors, and organized sections for better readability and impact.
🔧 Key Features:
Dual CV generation (ATS + Modern)
Real-time preview while editing
Structured form inputs for accuracy
Export options (PDF / print-ready)
Responsive design for all devices
🎯 Objective:

To bridge the gap between machine-readable resumes and visually attractive CVs, increasing users’ chances of passing automated screenings and impressing recruiters.
